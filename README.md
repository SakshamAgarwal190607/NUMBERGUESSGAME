# 🎯 Number Guessing Game (Python)

A simple and interactive command-line Number Guessing Game built using Python. This project is designed for beginners to practice core programming concepts like loops, conditionals, and user input handling in a fun way.

---

## 🚀 Features

- 🎲 Random number generation
- 🔁 Continuous guessing until correct answer
- 📉 Smart hints (Too High / Too Low)
- 🏆 Winning message on correct guess
- 💡 Clean and beginner-friendly logic

---

## 🧠 How It Works

- The program randomly selects a number within a defined range.
- The user is asked to guess the number.
- After each guess:
  - If the guess is higher → "Too High"
  - If the guess is lower → "Too Low"
- The game continues until the correct number is guessed.

---

## 🛠️ Tech Stack

- Python 🐍
- Built-in `random` module
